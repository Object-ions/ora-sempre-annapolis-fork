# Ora Sempre Annapolis

### By Moses Atia

## Description

This project is a website designed and developed for a skincare and spa business.

## Technical Stack

- HTML
- SCSS
- JavaScript / React.
- NoSQL
- Leaflet
- Firebase / Firestore
- Node.js
- npm
- Webpack

## Features

- Responsive design across various devices and screen sizes.
- Interactive map integration: Leaflet.
- User Authentication: Secure sign-up and sign-in functionalities powered by Firebase.
- Real-time Database: Firestore NoSQL database integration.
- Elegant UI/UX.
